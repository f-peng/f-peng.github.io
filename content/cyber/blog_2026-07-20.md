## La semaine en cybersécurité : WordPress et nginx en alerte maximale, SharePoint sous pression

Les dernières 48 heures ont été marquées par l’émergence de vulnérabilités critiques dans des solutions largement déployées, avec un focus particulier sur **WordPress** et **nginx**, tandis que la **CISA** sonne l’alarme pour des failles activement exploitées dans **SharePoint**. Les acteurs de la cybersécurité doivent prioriser les correctifs pour ces composants, dont certains permettent une exécution de code à distance sans authentification.

## Conclusion

Cette semaine confirme une tendance lourde : les **solutions web grand public** (WordPress, nginx) et les **outils collaboratifs** (SharePoint) restent des cibles privilégiées, avec des vulnérabilités permettant des attaques à distance et sans authentification. La **CISA** impose désormais des délais serrés pour les correctifs, soulignant l’urgence opérationnelle. Les équipes doivent également surveiller les **fuites de données sensibles** (comme le PIN de session dans Apple Passwords), souvent sous-estimées mais critiques pour la chaîne de confiance. Priorité absolue : appliquer les patches pour **CVE-2026-63030**, **CVE-2026-42533** et les failles SharePoint **KEV**.

<h3 class="class_h3">CVE-2025-24169 : fuite du PIN de session dans Apple Passwords sur macOS via os_log</h3>
- 📅 Mon, 20 Jul 2026 00:00:00 +0000
- 🔗 Source: https://cyberveille.ch/posts/2026-07-20-cve-2025-24169-fuite-du-pin-de-session-dans-apple-passwords-sur-macos-via-os-log/

<h2 id="-contexte">🔍 Contexte</h2>
<p>Publié le 13 juillet 2026 sur le blog de Josh Parnham, cet article est un writeup technique détaillé de <strong>CVE-2025-24169</strong>, une vulnérabilité affectant <strong>macOS Sequoia</strong> dans le composant natif de messagerie du gestionnaire de mots de passe Apple Passwords. La vulnérabilité a été corrigée à partir de <strong>macOS 15.3</strong>.</p>
<h2 id="-vulnérabilité">🐛 Vulnérabilité</h2>
<p>Le binaire <strong>PasswordManagerBrowserExtensionHelper</strong> (situé dans <code>/System/Cryptexes/App/System/Library/CoreServices/</code>) est utilisé par les extensions de navigateur iCloud Passwords pour Chrome et Firefox via le protocole <strong>native messaging</strong>. Lors de l&rsquo;établissement d&rsquo;une session SRP (Secure Remote Password), ce binaire journalisait le <strong>PIN de session en clair</strong> dans le journal système macOS via <code>os_log</code> avec le spécificateur <code>%{public}@</code>.</p>
